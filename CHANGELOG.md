# Changelog

## 1.0.0

Initial stable release.

* Added polygon split into 2 parts by target area.
* Added polygon equal-area division.
* Added polygon area-list division with remainder.
* Added line split by target length.
* Added line equal-length division.
* Added line length-list division with remainder.
* Preserves attributes and edits the active editable layer directly.
* Added plugin icon, metadata, README, and LICENSE for release packaging.
* Complete temporary polygon engine.
* Analytical active segment interpolation.
* Topology-safe polygon construction.
